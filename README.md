**##** Run Locally

Clone the project

```
bash

git@github.com:mskmee/rss_react.git
```

Go to the project directory

```
bash

cd rss_react
```

```
bash

cd rss_react
```

Install dependencies

```
bash

npm i
```

Start the server

```
bash

npm run start
```

Tests coverage

```
bash

npm run coverage
```

**##** Build

To build this project run

```
bash

npm run build
```

**##** Running Tests

To run tests, run the following command

```
bash

npm run test
```
